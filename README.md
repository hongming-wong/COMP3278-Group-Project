To run front-end:
1. CD into front-end and type "yarn install"
2. Then, type "yarn start"


To run the back-end:
1. Remember to check if you have flask installed. If not, run:
      - pip install flask
      - pip install -U flask-cors
2. CD into back-end
2. Type "python app.py"
